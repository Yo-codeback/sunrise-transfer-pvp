# 更新日誌 (CHANGELOG)

## 0.2.0-alpha (2025/11/13)

### 新增功能
- ✨ **等待期間無敵保護系統**：玩家在等待或倒數期間不會受到傷害
- ✨ **退出遊戲物品**：進入等待時給予紅色床物品，右鍵可退出遊戲
- ✨ **自動補血系統**：進入等待時自動補滿血量和飽食度
- ✨ **優化遊戲流程**：等待時不給Kit，遊戲開始時才給予完整Kit

### 修復問題
- 🔧 修復NPC生成問題：延遲載入確保世界已載入完成
- 🔧 優化遊戲狀態管理：更清晰的等待/倒數/開始流程

### 技術改進
- 🔧 改進傷害事件處理：等待和倒數期間取消所有傷害
- 🔧 改進物品管理：分離等待物品和遊戲Kit物品
- 📝 更新文檔：合併所有說明文檔到README.md

## 詳細更新說明

### 0.2.0-alpha 詳細說明

#### 遊戲等待系統優化

**等待期間保護機制**：
- 玩家加入遊戲後進入等待狀態
- 自動補滿血量和飽食度
- 獲得退出遊戲物品（紅色床，放在第9個槽位）
- 處於無敵狀態，不會受到任何傷害

**遊戲開始流程**：
- 達到人數上限後開始倒數3秒
- 倒數期間仍然無敵
- 倒數結束後遊戲正式開始
- 給予完整的Kit物品
- 移除退出物品
- 取消無敵效果，可以正常戰鬥

**退出遊戲方式**：
- 右鍵點擊退出遊戲物品（紅色床）
- 使用指令 `/pvp game leave` 或 `/pvp lobby`

#### NPC生成修復

- 修復NPC在插件啟動時無法正常生成的問題
- 添加延遲載入機制（延遲2秒），確保世界已完全載入
- 改進錯誤處理和日誌輸出

#### 文檔更新

- 合併所有分散的文檔到主README.md
- 更新版本號和日期
- 添加詳細的功能說明和使用指南
